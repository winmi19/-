#### 一、桌面管理类：

Ctrl + win + D = 新建桌面

Ctrl + win + ← = 切换至前一个桌面

Ctrl + win + → = 切换至下一个桌面

Ctrl + win + F4 = 关闭当前桌面

win + Tab = 选择桌面（任务视图）

  

#### 二、窗口管理类：

Alt + Tab = 选择窗口

Ctrl + Tab = 窗口内切换

win + Tab = 窗口总览

win + D = 最小化所有窗口（重新按一下显示桌面）

win + ↑ = 最大化当前窗口

win + ↓ = 还原当前窗口

win + ← = 将当前窗口置于左侧

win + → = 将当前窗口置于右侧

Alt + F4 = 关闭（删除）当前窗口

Ctrl + F4 / W = 关闭当前窗口内标签页（仅限部分应用）

Ctrl + Shift + Esc = 打开任务管理器


#### 三、文本 / 文件编辑类：

Ctrl + C = 复制（需选中文本）

Ctrl + X = 剪切（需选中文本）

Ctrl + V = 粘贴

Ctrl + Z = 撤销

Ctrl + Y = 还原

Ctrl + A = 全选

win + V = 打开剪切板

Delete = 删除文件（至回收站）/ 删除文本

Shift + Delete = 永久删除文件（不可恢复）/ 删除文本（可撤销）

  

#### 四、截屏 / 录屏类：

Alt + PrtScr¹ = 截屏当前窗口

win + PrtScr = 截屏完整屏幕

win + Shift + S = 可控制区域的截屏

Alt + A =微信截屏

Ctrl + Alt + A = QQ² 截屏

Ctrl + Alt + S = QQ 录屏





#### 五、系统辅助类：

win + Q = 全局搜索

win + R = 全局搜索并打开文件夹 （即小方框）

win + 1~9 = 打开 / 最小化任务栏上的第 1~9 个应用，若没有则不反应

win + 0 = 打开 / 最小化任务栏上的第 10 个应用，若没有则不反应

win + A = 打开系统通知栏

win + I = 打开系统设置

win + H = 在线语音识别（听写）

win + L = 锁屏

win + E = 打开文件资源管理器

win + U = 打开显示器设置

win + =² = 打开放大镜 / 放大光标所在区域

win + - = 缩小光标所在区域

  

#### 六、其他常用快捷键：

Ctrl (+ Shift ) + Space¹ = 切换中英文

Ctrl + Shift + F = 切换简体中文 / 繁体中文

F2 = 重命名选中文件

F11 = 打开 / 关闭网页全屏

Ctrl + =² = 放大窗口 / 网页大小

Ctrl + - = 缩小窗口 / 网页大小

Ctrl + 0 = 恢复窗口 / 网页至默认大小

Ctrl + P = 打印

Ctrl + F = 查找相关内容

Ctrl + B = 将选中文本**加粗** ³

Ctrl + U = 将选中文本添加下划线 ³

Ctrl + I = 将选中文本变为_斜体_ ³

Ctrl + E = 将当前行居中 ⁴

Ctrl + K = 添加超链接 ⁴

Ctrl + H = 查找和替换 ⁴

Ctrl + R = 刷新当前窗口 / 网页

Ctrl + S = 保存当前文件

Ctrl + O = 使用当前应用打开文件

Ctrl + N = 新建文件 / 窗口

Ctrl + R / L = 向左 / 右填补 ⁵

Ctrl + / = 缩小 / 放大字体 ⁵

注释：

1 [ Space ] 指空格键。

2 [ = ] 这里的 “=” 是指**按键**！

3 该功能仅适用于可编辑文本格式的软件。

4 该功能仅适用于**部分**可编辑文本格式的软件。
